Common Errors

This page contains common errors users face while sideloading.

• An internet connection is required to...

In some cases, when using enterprise certificates (or third party certificate providers), Apple will blacklist you from using that specific certificate. Developers have found workarounds to remove blacklists, but are only on supported versions.

• The app intergrity cannot be verified...

The certificate being used to install the app is revoked. Unless you have Appsync-Unified installed, this error will pop up. You must use another certificate. If using certificates from third parties (developer certificates), it is advisied you speak with your provider.

• Untrusted (Enterprise) Developer

All you need to do is open the Settings app > General > VPN & Device Management. Then, tap the enterprise/developer certificate, and click Verify App.
