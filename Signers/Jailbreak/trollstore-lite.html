TrollStore Lite

TrollStore Lite is an app signer that uses a built-in codesign bypass for jailbroken iOS only.

It has the same exact features of normal TrollStore, except that all applications installed with TrollStore Lite will only launch in a jailbroken state.

You MUST jailbreak in order to use TrollStore Lite.

Prerequisites

  • Jailbroken iPad & iPhone running iOS 14 - 18.2
  • https://havoc.app repo added into your package manager (Sileo, Zebra, Cydia, etc)

    • Adding Havoc

    1. Open your package manager.
    2. Click "Sources", or "Repos".
    3. At the + button, add
      https://havoc.app Copied!

      4. Installing TrollStore Lite

    4. In the search bar, find "TrollStore Lite".
    5. Click on the package, and tap Install.
