TrollMisaka

TrollMisaka is a type of installation vector that installs TrollStore on your device.

TrollStore is an app-permasigner that bypasses Apple's regular restrictions due to a coretrust (codesigning) exploit. Therefore, it allows unlimited apps, no revokes, JIT, arbitary entitlements, and much more.

TrollStore is not a jailbreak.
TrollMisaka is extremely unreliable. TrollMisaka is only for A8(X) iPads on iOS 15.7.2 - 15.8.4. If you are NOT on those versions, please use another TrollStore installer. You can find applicable installers at each "Get Started" iOS version, if supported. If you can only use TrollMisaka, you can try using other methods, such as: Installing Trollstore with Dopamine requires a jailbreak, can unjailbreak after installing safely) Installing TrollRestore(computer required) Proceed with this guide if you would like to use TrollMisaka (only if you cannot disable Find My or cannot jailbreak)

Prerequisites

  • A computer running Windows 7 or MacOS 10.12 or above
  • iPad, iPhone or Apple TV running iOS 7 or above
  • Latest version of Sideloadly.
  • Latest version of iTunes (not from the Microsoft store)
  • Latest version of iCloud (not from the Microsoft store)
  • A working Apple ID

Enabling Developer Mode

Starting iOS/iPadOS 16+, all users are required to enable developer mode to install developer apps.

  1. Open Settings.
  2. Scroll down to Privacy & Security.
  3. On Privacy & Security, scroll down to Developer Mode. Click enable.
  4. Your device may restart. Once enabled, proceed with the next steps.

Running Sideloadly

  1. Run the sideloadly installer.
  2. Once sideloadly has finished installing, open the application.
  3. Connect your iPhone or iPad to your computer. Ensure the device is trusted and unlocked.
  4. Drag & drop the .ipa file, or select it.
  5. Enter your Apple ID credentials. Your login details will only be sent to Apple for verification and are not tracked by AltServer.
  6. Tap "Install".

Verifying Application

  1. Open Settings.
  2. Go to General.
  3. Scroll to VPN & Device Management.
  4. Open it and you will find a developer application.
  5. Click on it. Once opened, click "Verify App".
