TrollMisaka

TrollMisaka is a type of installation vector that installs TrollStore on your device.

TrollStore is an app-permasigner that bypasses Apple's regular restrictions due to a coretrust (codesigning) exploit. Therefore, it allows unlimited apps, no revokes, JIT, arbitary entitlements, and much more.

TrollStore is not a jailbreak.

TrollMisaka is extremely unreliable.

TrollMisaka is only for A8(X) iPads on iOS 15.7.2 - 15.8.4. If you are NOT on those versions, please use another TrollStore installer. You can find applicable installers at each "Get Started" iOS version, if supported.

If you can only use TrollMisaka, you can try using other methods, such as:

• Installing Trollstore with Dopamine requires a jailbreak, can unjailbreak after installing safely)

• Installing TrollRestore(computer required)

Proceed with this guide if you would like to use TrollMisaka (only if you cannot disable Find My or cannot jailbreak)

You must use AltStore, SideStore or Sideloadly, or any signer that uses developer certificates. This is because enterprise certificates do not support entitlements to add custom keyboards.

Prerequisites

  • A computer running Windows 7 or MacOS 10.12 or above
  • iPad or iPhone running iOS 15.7.2 - 15.8.4
  • Latest version of Sideloadly.
  • Latest version of iTunes (not from the Microsoft store)
  • Latest version of iCloud (not from the Microsoft store)
  • Latest version of TrollMisaka.
  • A working Apple ID

Running Sideloadly

  1. Run the sideloadly installer.
  2. Once sideloadly has finished installing, open the application.
  3. Connect your iPhone or iPad to your computer. Ensure the device is trusted and unlocked.
  4. Drag & drop the .ipa file, or select it.
  5. Enter your Apple ID credentials. Your login details will only be sent to Apple for verification and are not tracked by AltServer.
  6. Tap "Install".

Verifying Application

  1. Open Settings.
  2. Go to General.
  3. Scroll to VPN & Device Management.
  4. Open it and you will find a developer application.
  5. Click on it. Once opened, click "Verify App".

Adding a keyboard

  1. Open TrollMisaka, then exit the application, leaving it in the app switcher. Do not close it.
  2. Open the Settings application.
  3. Head to:
    General > Keyboard > Keyboards > Add new keyboard
  4. Tap TrollMisaka, then toggle on Allow Full Access, then tap Allow.
  5. Re-open TrollMisaka.

    6. After reopening TrollMisaka, open the app switcher, then reboot your device via hardware keys.

Injecting TrollStore Helper

This process may take several tries.
  1. Open & unlock your device.
  2. Immediatly once unlocked, open Spotlight Search (you can do this by swiping down on the middle of the home screen.)
  3. Tap the globe icon on keyboard.

    4. • If required, select TrollMisaka from the list of keyboards.

  4. Tap kopen.

    5. • If the device reboots, try these steps again.

  5. Tap Install Trollstore.
  6. • If the keyboard refreshes, try again. If the issue persists, try again in Safari instead.

Installing TrollStore

  1. Re-open the app switcher.
  2. Open the TrollMisaka app.
  3. Tap Install TrollStore.

Installing Persistence Helper

  1. Open the TrollStore application on your home screen.
  2. Press Settings, then press Install Persistence Helper..
  3. Select a random application you don't use. If you don't know what to pick, choose Tips.
