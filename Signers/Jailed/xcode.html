Get Started

XCode

XCode is an app on MacOS which can be used to develop apps, and more. One of the features that XCode has is sideloading apps with your Apple ID.

XCode should only be used to sideload if you are sideloading via tvOS, watchOS or visionOS. Otherwise, alternatives like Sideloadly exist for iOS.
This guide is currently a stub. Some information may be incorrect or inaccurate. Please visit site information to learn about contributing.

Prerequisites

  • A computer running MacOS
  • Latest version of XCode
  • Latest version of iDeviceInstaller.
  • A working Apple ID (registered as a developer)

Enabling Developer Mode

Starting iOS/iPadOS 16+, all users are required to enable developer mode to install developer apps.

  1. Open Settings.
  2. Scroll down to Privacy & Security.
  3. On Privacy & Security, scroll down to Developer Mode. Click enable.
  4. Your device may restart. Once enabled, proceed with the next steps.

Opening XCode

  1. Launch XCode.
  2. Select "Create a new XCode project"
  3. Select a template "Single View App" and tap "Next".
  4. Fill in all the details and click "Next".
  5. Save the project to a memorable location. You will need this later.

Connecting your iOS device

  1. Connect your device to your computeer.
  2. Select your device in the list of available devices in XCode.

Preparing Signature

  1. Go over to the project settings by clicking on the project name in the XCode sidebar.
  2. Under the "Signing & Capabilities" tab, select your Apple ID from the "Team" dropdown menu.
  3. If not already, check "Automatic Manage Signing".

Launching Termjinal

  1. Open the terminal.
  2. Run "cd /" to where your .ipa file is.
  3. Run `shell
  4. Run ideviceinstaller -i YourAppName.ipa

Verifying Application

  1. Open Settings.
  2. Go to General.
  3. Scroll to VPN & Device Management.
  4. Open it and you will find a developer application.
  5. Click on it. Once opened, click "Verify App".
