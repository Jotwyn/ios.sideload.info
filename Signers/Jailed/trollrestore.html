TrollRestore

TrollRestore is a type of installation vector that installs TrollStore on your device.

TrollStore is an app-permasigner that bypasses Apple's regular restrictions due to a coretrust (codesigning) exploit. Therefore, it allows unlimited apps, no revokes, JIT, arbitary entitlements, and much more.

TrollStore is not a jailbreak.
Make sure to disable Find My before proceeding with the installation.

Please select your OS. (Windows selected by default)

Prerequisites

  • An iPad or iPhone running iOS 15.2 - 16.7 RC & 17.0
  • Latest version of TrollRestore
  • Computer running Windows 10 or above

Installing TrollRestore (Windows)

  1. Plug your iOS device into your computer. Ensure your iDevice is trusted.
  2. Open the File Explorer program.
  3. Find TrollRestore, and double click it. The download should be most likely in your Downloads folder unless specified previously.
  4. When asked for a system applicaction, type in any installed system application you want to override. Note that the app will no longer be usable until reset.

Installing TrollRestore (MacOS)

  1. Ensure to unzip the file before prcoeeding.
  2. Plug your iOS device into your computer. Ensure your iDevice is trusted.
  3. Open the Finder window.
  4. Find TrollRestore, and double click it. The download should be most likely in your Downloads folder unless specified previously.
  5. When asked for a system applicaction, type in any installed system application you want to override. Note that the app will no longer be usable until reset.

Prerequisites

  • An iPad or iPhone running iOS 15.2 - 16.7 RC & 17.0
  • Latest version of TrollRestore
  • Computer running Linux

Installing TrollRestore (Linux)

  1. Open a terminal app on your computer.
  2. Run
    cd path/to/TrollRestore
  3. Run
    pip3 install -r requirements.txt
    and follow the on-screen instructions.
  4. Plug your iOS device into your computer. Ensure your iDevice is trusted.
  5. When asked for a system applicaction, type in any installed system application you want to override. Note that the app will no longer be usable until reset.
